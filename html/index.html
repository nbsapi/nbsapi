NBSAPI

Common data exchange mechanism for tools built for Nature-Based Solutions

What is NBSAPI?

NBSAPI is a API specification, designed to provide data and resources for sharing and exchanging data around nature-based solutions. With a focus on adaptation targets like flooding, drought, heat, and groundwater management, the API empowers developers to create interoperable applications.

Key Features

NBSAPI provides community and tools to enable futr

  • Compatible with your application, easy to implement and understand
  • Real-time adaptation target tracking
  • Flexible filtering and query options
  • Easy integration with FastAPI and other frameworks
