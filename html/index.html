About NBSAPI

NBSAPI is designed to provide data and resources for nature-based solutions. With a focus on adaptation targets like flooding, drought, heat, and groundwater management, our API empowers developers to create sustainable and resilient applications.

Key Features

  • Comprehensive data on nature-based solutions
  • Real-time adaptation target tracking
  • Flexible filtering and query options
  • Easy integration with FastAPI and other frameworks
